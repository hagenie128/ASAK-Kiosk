# 상세 워크로그 (entries)

기능·이슈·PR 단위의 **상세** 작업 기록을 **담당자별**로 둡니다.

## daily와의 관계

| | `daily/{이름}/YYYY-MM-DD.md` | `entries/{이름}/` (이 폴더) |
|---|---|---|
| 단위 | 하루 | 기능·이슈 하나 |
| daily 역할 | **오늘 요약** 표 + **오늘 작업** 미니 카드 + 이 파일 링크 | — |
| entries 역할 | — | [03-work-log-template.md](../../docs/guides/03-work-log-template.md) 12섹션 전체 |

## 작성 규칙

1. [`docs/guides/03-work-log-template.md`](../../docs/guides/03-work-log-template.md)를 복사해 시작
2. 경로: `entries/{이름}/YYYY-MM-DD-짧은-설명.md`
3. 같은 날짜 [`daily/{이름}/`](../daily/) — 표 + 미니 카드 + 이 파일 링크
