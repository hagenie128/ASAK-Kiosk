# 상세 워크로그 (entries)

기능·이슈·PR 단위의 **상세** 작업 기록을 둡니다.

## 작성 규칙

1. [`docs/guides/03-work-log-template.md`](../../docs/guides/03-work-log-template.md)를 복사해 시작
2. 파일명: `YYYY-MM-DD-짧은-설명.md` (예: `2026-07-05-menu-option-ui.md`)
3. 같은 날짜의 [`daily/`](../daily/) 파일 표에 한 줄 요약 + 이 파일 링크 추가
