## Week 5 MVP (SCR-001~008) + Week 6 (SCR-009~011)

- **SCR-001** 홈 화면 — ASAK 키오스크 시작 화면. 브랜드명과 주문 시작 버튼을 보여준다.
- **SCR-002** 먹고가기 / 포장 선택 — 고객이 주문 유형을 먹고가기 또는 포장으로 선택하는 화면
- **SCR-003** 메뉴 선택 — 카테고리별 ASAK 메뉴 목록을 카드형으로 보여주는 화면
- **SCR-004** 메뉴 상세 / 옵션 선택 — 메뉴 상세 정보를 확인하고 기본 재료 제외, 드레싱/토핑 등 옵션을 선택하는 화면입니다.
- **SCR-005** 장바구니 — 고객이 담은 메뉴, 옵션, 수량, 총 금액을 확인하고 수정하는 화면
- **SCR-006** 주문 확인 — 결제 전 주문 메뉴, 옵션, 수량, 주문유형, 총 금액을 최종 확인하는 화면
- **SCR-007** 결제 — 주문 금액을 확인하고 실제 PG 연동 없이 가상 결제를 진행하는 화면
- **SCR-008** 주문 완료 — 결제 완료 후 주문번호와 주문 접수 상태를 보여주는 화면
- **SCR-009** 관리자 주문 관리 — 관리자가 주문 목록을 확인하고 특정 주문의 상세 화면으로 이동하는 화면
- **SCR-010** 관리자 주문 상세 — 관리자가 특정 주문의 메뉴, 옵션, 주문유형, 결제상태, 주문상태를 확인하는 화면
- **SCR-011** 관리자 판매 항목 품절 관리 — 관리자가 메뉴 또는 재료의 품절 상태를 변경하는 화면입니다.
